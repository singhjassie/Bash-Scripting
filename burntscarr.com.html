Seedr To Google Drive Free

Getting a file or folder from Seedr to Google Drive is free and easy with just a little code to learn!

Speaking of code, if you haven't already, go make a new notebook on Google Colab and we'll begin below!

Setting up your notebook in Google Colab

  1. Go to the Google Colab site, and click new notebook. This isn't required if you've done so already.
  2. Name the notebook. Something like "Seedr". To do this, click the "Untitled.ipynb" up at the top left, and rename to "Seedr.ipynb"
  3. Mount your Google Drive. Click the folder icon on the left of the screen, then click the 3rd icon under "Files" that resembles. a folder with the Google Drive icon on it.
  • This will either automatically mount your google drive, pop up a prompt asking you if you want to, or simply generate a code cell that you need to run.
  • If a pop up appears, choose to connect to your Drive.
  • If a code cell appears, even after choosing to connect to your drive, play the cell and click the link it gives you, then copy the token it gives you after you log in back into the generated code cell in Colab (a little box will appear if you click under the text if it isn't already showing, watching the video helps see what I'm talking about.)
  1. In a new code cell (there may already be an empty one) you'll type %cd /content/drive/MyDrive/ and press the play button. It should read out /content/drive/MyDrive/ after that. (This will put you into your main Google Drive folder. Know that if you don't have free space for the file that you are about to copy, it will fail)
  2. Now create a new code cell with the "+ Code" button on the top left. Then prepare a cell with !wget "" -O "" you'll need this later.

Getting your file into Seedr

If you're a Seedr user, you should already know this part. However, just incase you're interested in how this works and are not already a Seedr user, here's how you'd begin.

  1. Create / Login to a Seedr Account
  2. Find the torrent you want to download. If you after movies, try RARBG. If you need TV Shows, RARBG has those too.
  3. Copy the torrent's magnet link, or download the torrent. For RARBG specifically, you'll see a little magnet icon next to the name of the torrent. The name of the torrent itself is a link to the .torrent file, and the magnet icon is the magnet link. If you right click it and copy the link (NOT the image link) you can paste it into Seedr's "Paste Link URL Here" box up top. Or you can press the + on the right side of that box and choose the .torrent file if you chose to download that.
  4. Let it download. As long as you see that it has started, it should finish. If it stays at 0% for longer than an hour, I would personally assume that it is a dead torrent (one without people uploading it).
  5. Copy the link. You can either right click the folder that contains the torrent's files and copy the download link from there, OR you can go inside the folder and right click the specific file you want to download and copy that file's download link. Remember which one you do! It'll be useful later.

Setting up the download code using the newly copied link

  1. Back in Step 5 of the Setup section, you made a code cell with !wget "" -O "" in it. You'll want to paste your link in the first set of quotations. !wget "LINK HERE" -O ""
  2. In the second set of quotations, you'll put the name of the file/folder you're downloading.
  • If you copied a folder link, you'll need to add .zip to the end of this name.
  • If you copied a file link, you'll need to add the same extension as the file.
  • Folder Example (Folder is named "The.One.2001.1080p.BluRay.x265-RARBG"):
  • !wget "LINK HERE" -O "The.One.2001.1080p.BluRay.x265-RARBG.zip" or I can modify it since I name my movies a different way !wget "LINK HERE" -O "The One (2001).zip"
  • This will download a file called The.One.2001.1080p.BluRay.x265-RARBG.zip (or whatever you named it) into your main Google Drive.
  • File Example (File inside of previous folder, called "The.One.2001.1080p.BluRay.x265-RARBG.mp4")
  • !wget "LINK HERE" -O "The.One.2001.1080p.BluRay.x265-RARBG.mp4"
  • This will download a file called "The.One.2001.1080p.BluRay.x265-RARBG.mp4" (or whatever you named it) into your main Google Drive.
  1. Press play and watch the progress bar.
  2. Check your GDrive for the newly added file.
